### Simple Project focused on Hospitals..

This is a **simple showcase project** designed to simulate a hospital system backend.

* Backend:
  * MongoDB - used as main database (quick development)
  * Docker - containerized services for consistent environments
  * Node.js with Express – for fast and clean RESTful API developme
  * **Mongoose** – for schema modeling and database interaction


### Features

* Patient registration and management
* Appointment scheduling (consultations)
* Modular and scalable architecture
* Environment-based configuration with `.env` or YAML
* MongoDB integration with proper pagination support
* RESTful APIs with clean separation of concerns
